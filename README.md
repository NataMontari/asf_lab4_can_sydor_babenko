Цей проект реалізує повну багатовузлову систему зв'язку **CAN (Controller Area Network)** з використанням мікроконтролерів **STM32 "Blue pill"**. Він організований у три незалежні модулі мікропрограми—**CAN_Node_A**, **CAN_Node_B** та **CAN_Node_C**—кожен представляє фізичний пристрій на шині CAN. Кожен вузол налаштований з власними ідентифікаторами повідомлень, фільтрами, перериваннями та операційною логікою, що дозволяє їм обмінюватися даними та реагувати на повідомлення в режимі реального часу.

Прошивка для кожного вузла будується за допомогою інструменту налаштування **STM32CubeMX** разом з бібліотеками **HAL і LL**, наданими STMicroelectronics, а також ST-Link. Ці бібліотеки обробляють ініціалізацію периферійних пристроїв CAN, GPIO, годинників і контролерів переривань. 

Кожен вузол CAN демонструє основні функції зв'язку:

* передача періодичних кадрів CAN,
* отримання повідомлень через апаратні фільтри,
* обробка переривань для завершених передач або нових вхідних кадрів,
* розбір корисного навантаження та ініціювання дій на основі вмісту повідомлення.

Три вузли призначені для взаємодії один з одним у спільній мережі CAN, утворюючи невелику розподілену систему.
